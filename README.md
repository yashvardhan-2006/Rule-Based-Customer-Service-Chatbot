# 🤖 Customer Service Chatbot

A simple rule-based chatbot that answers common customer queries using 
keyword pattern matching. Built as part of my AI internship at Codec Technologies.

## Features
- Handles 11 intents — orders, refunds, payments, account, complaints & more
- Instant responses using JavaScript RegEx pattern matching
- Typing animation, suggestion chips & help command
- No installation needed — just open chatbot.html in any browser

## Tech Stack
HTML · CSS · Vanilla JavaScript · RegEx

## How to Run
1. Clone the repo
2. Open chatbot.html in your browser — done!
